UX/UI Design

“Design isn’t finished until somebody is using it.”

– Brenda Laurel

The web design world is constantly evolving. The web design not only emphasizes a beautiful layout and design, but also to value more experience that users can obtain through services and products. This requires different perspectives and skills from conventional design.

Upon designing UI/UX, the main goal is to have a functional format for the audience. To elaborate the target audience will be provided with specific set information on products (materials, delivery, etc.) and set goals.


Project example

Aloha Collections Website



Client: Aloha Collections
Service: Website Design
Software: Adobe XD

The project is to assist a local group with designing their online shopping website. This would be user friendly and appealing in both functionality and design to the Japanese tourists. Since they will be able to have easy access to products and any further information sold from local artists.
